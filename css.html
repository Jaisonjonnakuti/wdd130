My Programming Journey

I've always been fascinated by how websites work and how they can reach people all around the world. Web development combines creativity with technical skills, allowing me to build something that others can interact with immediately. The instant feedback loop of making changes and seeing results is incredibly satisfying.

HTML provides the structure for all web pages and is the foundation of web development. CSS allows me to style elements and create visually appealing layouts. I'm also learning JavaScript to add interactivity and dynamic features to my websites.

Other technologies I'm excited to learn include:

  • React for building interactive UIs
  • Node.js for server-side JavaScript
  • CSS frameworks like Bootstrap and Tailwind

By the end of this year, I aim to:

  • Build a complete portfolio website
  • Create a responsive web application
  • Learn a JavaScript framework
  • Contribute to an open-source project

Long-term, I'd like to work as a full-stack developer creating meaningful applications that solve real problems.
